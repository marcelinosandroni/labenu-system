Labenu System API Version 1.0

End Points

Class

  • GET /class list all classes
  • GET /class?name=cruz search class by name
  • POST /class create a new class
  • GET /class/:name list class details
  • PUT /class/:name update class
  • DELETE /class/:name delete class
  • GET /class/:name/students list all students in class
  • GET /class/:name/teachers list all teachers in class

Students

  • GET /student list all students
  • GET /student?name=mar&age=28 search student by name or age
  • POST /student create a new student
  • GET /student/id list student details
  • PUT /student/id update student
  • DELETE /student/id delete student
  • GET /student/id/class get class of student
  • PUT /student/id/class change the student class
  • DELETE /student/id/class remove student from current class
  • GET /student/students list student hobbies
  • POST /student/id/hobbies add new hobbie for student
  • PATCH /student/id/hobbies remove a hobbie from student
  • DELETE /student/id/hobbies delete all student hobbies
  • GET /student/id/teachers list all teachers of the student

Teachers

  • GET /teachers
  • POST /teachers create a new teacher
  • PUT /teachers/id update teacher
  • DELETE /teachers/id delete teacher

Hobbies

  • GET /hobbies
  • POST /hobbies create a new hobby
  • PUT /hobbies/id update hobby
  • DELETE /hobbies/id delete hobby
  • GET /hobbies/name list all students in hobbies

Skills

  • GET /skills
  • POST /skills create a new skill
  • PUT /skills/id update skill
  • DELETE /skills/id delete skill
  • GET /skills/name list all teachers in skills
